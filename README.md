# 🚀 AIPromptify

AIPromptify is a **full-stack AI Prompt Sharing & Marketplace Platform** where users can create, discover, share, bookmark, and manage AI prompts for popular AI tools like ChatGPT, Gemini, Claude, and Midjourney.

This platform helps users find high-quality prompts to improve productivity, creativity, automation, and AI-powered workflows. Users can publish prompts publicly, save favorite prompts, review prompts, and unlock premium private prompts through secure payment integration.

## 📌 Features

- 🔐 Secure authentication using Better Auth and JWT
- 👥 Role-based access control (User, Creator, Admin)
- 🧠 AI Prompt marketplace with search, filter, bookmark, and review system
- 💎 Premium private prompts unlocked via Stripe payment integration
- 📊 Analytics dashboard for prompt performance and growth
- 🛠 Admin dashboard for managing users, prompts, payments, and reports
